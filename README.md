## Automated-report-generator
Project Overview:

The Automated Report Generator is a Python-based application that reads data from an external file, performs basic data analysis, and automatically generates a well-formatted PDF report.
This project demonstrates automation, file handling, data analysis, and PDF generation using Python.

Objectives:

1. Read structured data from a file (CSV)
2. Analyze the data (count, average, min, max)
3. Generate a professional PDF report automatically
4. Reduce manual effort in report creation

Technologies Used:
1. Python3
